Soluciones para enfrentar emergencia COVID-19
  • Proyectos
  • Cómo colaborar
  • Noticias
  • Contacto

Estamos comprometidos con la sociedad y estamos utilizando nuestras capacidades, tiempo y talentos para buscar soluciones durante esta emergencia. Queremos ayudar a reducir los contagios, aumentar la seguridad del personal sanitario, apoyar en la toma de decisiones y proponer soluciones técnicas para el tratamiento. En conjunto con la Facultad de Medicina, el Centro de Innovación y otras facultades de la UC, más la colaboración de otras instituciones estatales y privadas, estamos impulsando los siguientes proyectos:

Cámara de aislación para atención de pacientes graves
Este dispositivo permite al personal médico protegerse de los aerosoles emitidos por un paciente contagiado.
Leer Mas
Cámara de luz UV para inactivación de covid19
Es una cámara que utiliza luz UV para degradar o desactivar el COVID-19 en superficie de instrumentos.
Leer Mas
Mascarillas de protección reciclables con nanotecnología
Mascarillas fabricadas con telas microporosas que permiten eliminar activamente el virus.
Leer Mas
Máscaras de protección para personal de la salud
Máscaras de producción de bajo costo y protección completa del rostro.
Leer Mas
Generador de ozono para desinfectar espacios
Este generador de ozono puede ser utilizado para desinfectar ambientes rápidamente cubriendo todo el volumen de la habitación.
Leer Mas
Análisis y predicciones sobre la evolución del COVID-19
Sistema que permiten realizar análisis y proyecciones sobre cómo evolucionará la enfermedad a partir de la información recopilada por el Ministerio de Salud.
Leer Mas
Asesoría de gestión de turnos en hospitales para enfrentar crisis sanitaria
Sistema que permite optimizar la gestión de turnos del personal de la salud en hospitales públicos para hacer frente a la sobredemanda por COVID-19.
Leer Mas
Conector y filtro para conversión de máscaras de uso recreativo
A través de este sistema se puede convertir máscaras de snorkel en máscaras protectoras para el personal médico de forma económica y segura.
Leer Mas
Soporte facial para aliviar a pacientes con ventilación mecánica invasiva
El dispositivo SOFA es un soporte facial para pacientes que debido a la dificultad que presentan para respirar deben ser tratados tendidos boca abajo.
Leer Mas
Accionador mecánico de AMBU
Este ventilador acciona mecánicamente una bolsa manual de respiración haciendo posible el flujo de aire en los pacientes afectados gravemente con el COVID-19.
Leer Mas
Desinfectante natural y biodegradable para diversas superficies
Solución líquida para limpiar materiales, superficies y recintos potencialmente contaminados con cargas virales patogénicos.
Leer Mas
Kit olfativo rápido para identificación temprana de asintomáticos con COVID-19
Actúa mediante la evaluación de tres aromas cuidadosamente seleccionados, entregando el estado olfativo de los pacientes que presentan COVID-19.
Leer Mas
Sistema para certificar capacidad de filtrado de mascarillas
Este sistema permite determinar la capacidad de filtrado de las mascarillas después de ser autoclavadas. 
Leer Mas
Sistema de registro de camas y reasignación de pacientes
Sistema que permite contar con información actualizada sobre la disponibilidad de recursos como camas y equipamiento en el sistema de salud público-privado.
Leer Mas
Ventilador Mecánico de Emergencia Rápido
Ventilador mecánico con intubación para ser usado en caso de emergencia en pacientes que ya no pueden respirar.
Leer Mas
Ventilador mecánico portátil
Este dispositivo es robusto y fácil de replicar, además permite diversos modos de ventilación para asistir a personas con dificultades respiratorias.
Leer Mas
